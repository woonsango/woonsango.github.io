## Git
- 버전 관리 시스템 중 하나
- Git은 로컬시스템에서 버전 관리 가능
- Git을 호스팅하는 원격시스템(Github)을 통해 다른 사람들과 협력 가능
- 데이터베이스를 구성하여 파일의 변경 기록을 관리

## Github
- Git 호스팅 사이트 중 하나
- 여러명의 개발자가 협업하기 용이
