프로젝트를 Build한 과정
=============

1. GitHub에서 Theme 다운 받기

    - 나같은 경우 https://github.com/sergiokopplin/indigo GitHub에서 indigo Theme을 다운 받았다.
    - 이후 나의 Git Local 저장소에 파일을 풀어줬다.
    - Theme이 잘 적용됐는지 확인하기 위해 **bundle exec jekyll serve** 명령어를 이용해 서버를 실행했다.
